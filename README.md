NAME:
   bookmark - Uma aplicação para salvar e organizar os favoritos direto pelo terminal

USAGE:
   bookmark [global options] command [command options]

        Cadastra uma nova collection ou um novo bookmark

        collection, c  criar nova coleção
        bookmark, b    criando um novo favorito

   list, l    Retorna uma lista de collection, bookmark
     usage:
        Permite listar uma coleção ou um bookmark
     
     commands:
        collection, c  Lista uma coleção
        bookmark, b    criando um novo favorito
        help, h        Shows a list of commands or help for one command
   update, u  Atualiza collection, bookmark
   delete, d  Deleta collection, bookmark
   get, g     Busca collection, bookmark
   help, h    Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h  show help
